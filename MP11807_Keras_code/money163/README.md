# newsCrawler


# 使用前需要先：
1. 安裝scrapy
2. 建立一個scrapy專案
3. 將相關程式複製到專案對應的目錄

# 主要特點

- 1. 可以自訂抓取某一天的網頁。
- 2. 抓取專案作為字典輸出到單個檔案，而非輸出至MongoDB，這樣更簡單。
- 3. 一些舊程式不適應新scrapy版本的地方進行修改，不一一列出。
- 4. 將股票代碼和新聞頁碼作為參數，可以靈活控制抓取的內容。

# 使用方法：
- 在..\ch04\ 根目錄下執行下列DOS命令：
--  ..\ch04>scrapy crawl stocknews -a id='600000' -a page='02'

- 在..\ch04\ 根目鍵下執行下列DOS命令，調用Python腳本，以抓取所有股票對應的歷史新聞：
--  ..\ch04>python PythonRun.py
